# Посмотри в окно

Интерактивный сайт, позволяющий «заглянуть в окно» разных городов и увидеть, как они выглядят в разное время суток.  
Проект создан в рамках учебного спринта по фронтенд‑разработке.

## 📌 Функционал
- Просмотр видео‑фрагментов из разных городов.
- Выбор времени суток: утро, день, ночь.
- Поиск города через форму.
- Кнопка «Показать ещё» для загрузки дополнительных карточек.
- Кастомные чекбоксы и кнопки с собственной стилизацией.
- Прелоадеры, которые появляются только поверх контейнеров с видео и списком карточек.

## 🛠️ Технологии
- **HTML5** — семантическая разметка.
- **CSS3** — стилизация, работа с псевдоэлементами, позиционирование, кастомные состояния `:focus-visible`.
- **Flexbox** — для позиционирования элементов.
- **Git/GitHub** — управление версиями и публикация проекта.
